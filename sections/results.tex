{
\setbeamerfont{itemize/enumerate body}{size=\small} % template pins lists to \normalsize
\begin{frame}{Income run}
  \framesubtitle{Open system}
  \centerline{\includegraphics[width=0.60\linewidth]{figures/population_faucet.png}}
  \small
  \begin{itemize}
    \item The 40--80 swings are \textbf{caused by the node cap}, as income is cut at 60 nodes and restored at 40.
    \item Each spawn spreads capital over more nodes, shrinking per-node surplus. At these node counts, the cap masks it (\(\to\) future work).
  \end{itemize}
  \textcolor{tud Orange}{\textbf{Introducing income makes the fleet persist past 510 days.}}
\end{frame}
}


{
\setbeamerfont{itemize/enumerate body}{size=\small} % template pins lists to \normalsize
\begin{frame}{Survival Distribution}
  \smallskip
  \centerline{\includegraphics[width=0.55\linewidth]{figures/lifespan_hist_faucet.png}}
  \small
  \begin{itemize}
    \item Uneven early income splits the fleet, a node receiving more income early keeps large surplus, so \textbf{early wealth compounds} into a few long-lived nodes.
  \end{itemize}

  \begin{block}{High turnover}
    Survival only needs \emph{some} node holding a surplus, so a well-funded
    minority eventually replaces the short-lived majority.
  \end{block}
\end{frame}
}

  \small
  \begin{columns}[T, onlytextwidth]

      \smallskip

      \smallskip

      \includegraphics[width=\linewidth]{figures/caution_alive_tertiles_faucet.png}
    \end{column}
    \begin{column}{0.5\textwidth}
      \smallskip
      \smallskip
      \smallskip
      Caution should be \textbf{reverting toward 0.5}, yet mid-caution drops while
      both extremes grow. Means \textcolor{tud Orange}{selection pressure} is stronger than the artificial center-pull.

      \smallskip
      \textbf{Hypothesis:} the income on/off cycle rewards opposite strategies.
      Low caution exploits growth, high caution survives
      drought, while mid caution benefits from neither.

      \smallskip
      \textbf{Two falsifiable predictions:}
      \begin{enumerate}
        \item low-caution reproduce faster during income.
        \item high-caution die less during pauses.
      \end{enumerate}
    \end{column}
  \end{columns}
\end{frame}

      \smallskip
      \begin{exampleblock}{Prediction 1 --- confirmed}
        Low-caution reproduces \(\sim\)48\% more with income
        (0.0294 vs.\ 0.0198 births/day).
      \end{exampleblock}
      \begin{alertblock}{Prediction 2 --- disproved}
         High-caution has no survival advantage (0.0088 vs.\ 0.0066 deaths per day). Thus, selection favored low caution, the reproduction gap widened with every phase.
      \end{alertblock}

      {\footnotesize Income pauses were \textbf{too short} for caution to pay off. In hindsight, income should have been resumed with only 10-20 nodes alive. Testing on better host machine \(\to\) future work.}
