    \smallskip
    \smallskip
  \centerline{\includegraphics[width=0.75\linewidth]{figures/population_faucet.png}}
  \small
  \begin{itemize}
    \item Income is cut at 60 nodes and restored at 40.

  \end{itemize}
  \textcolor{tud Orange}{\textbf{Introducing income makes the fleet persist past 510 days.}}

{
\setbeamerfont{itemize/enumerate body}{size=\small} % template pins lists to \normalsize
\begin{frame}{Survival Distribution}
  \smallskip
  \centerline{\includegraphics[width=0.55\linewidth]{figures/lifespan_hist_faucet.png}}
  \small
  \begin{itemize}
    \item Uneven early income splits the fleet, a node receiving more income early keeps large surplus, so \textbf{early wealth compounds} into a few long-lived nodes.
  \end{itemize}

  \begin{block}{High turnover}
    Survival only needs \emph{some} node holding a surplus, so a well-funded
    minority eventually replaces the short-lived majority.
  \end{block}
\end{frame}
}

      Caution should be \textbf{reverting to 0.5}. Observing the opposite means \textcolor{tud Orange}{selection pressure} is stronger than the artificial center-pull.

      \medskip
      \medskip
      \medskip
      \begin{block}{Secondary Hypothesis}
          The income cycle rewards opposite strategies. Low caution exploits growth, high caution survives drought, while mid caution benefits from neither.
      \end{block}

    \end{column}
  \end{columns}
\end{frame}
